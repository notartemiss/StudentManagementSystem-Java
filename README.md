Student managerment system - Java 

FEATURES  -
>add students
>view all students
>search students by ID
>update students marks
>delete student


SQL Table 
run this sql -
CREATE DATABASE student_db;

CREATE TABLE students (
    id INT PRIMARY KEY AUTO_INCREMENT,
    name VARCHAR(100),
    email VARCHAR(100) UNIQUE,
    marks INT
);
